Pisces: A P2P Meta-Grid System

 

To support various Grid services to satisfy with heterogeneous needs, the Grids integrated with distinct types of services provided by spreading resources across the network. This project will develop the hierarchical architecture to coordinate spreading Grid resources. For the purposed of developing distributed autonomic Grid systems, the centralized coordination architecture is utilized to manage the intra-Grid resources and the Peer-to-Peer (P2P) technology is applied to communicate with inter-Grid systems. The Pisces project will exploit Grid system management and P2P technology to construct P2P computing Grid system and P2P data Grid system with the consideration of integrating with two types of Grid resources: computing resources and storage resources. The Grid system information could be integrated by developing the information monitoring system. With exploiting the computing resource management system and the data storage management system, resource accessing in Grid system could be super intended. Moreover, rely on metadata catalog management system and P2P middleware to communicate with distributed autonomic Grid systems and to support the developing mechanism for resource discovery and metadata catalog lookup. Based on the aforementioned techniques, this project will be applied P2P technology to Grid system to construct a P2P Meta-Grid system. The Pisces project could not only achieve cross-Grid systems and improve the scalability, but also provide a platform for large-scale application development and testing.
